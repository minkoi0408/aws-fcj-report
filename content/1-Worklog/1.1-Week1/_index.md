### Week 1 Achievements:

* **AWS Foundational Knowledge:**
  * Clearly understand AWS and grasp the function of the 4 core service groups:
    * **Compute:** Provides processing resources for applications (virtual machines, containers, etc.).
    * **Storage:** Used for data persistence, backup, and recovery.
    * **Networking:** Manages network infrastructure, security, and connectivity between resources.
    * **Database:** Offers managed services for relational (RDS) and non-relational databases.
* **Identity and Access Management (IAM):**
  * Successfully created, configured, and identified the AWS Free Tier account.
  * Proficient in creating and managing **Group Users** and **Users**.
  * Understand the IAM login mechanism, where Users within the same Group share the assigned permissions.
* **Operational Proficiency (Console & CLI):**
  * Familiarized with the **AWS Management Console** and know how to find, access, and use services via the web interface.
  * Completed installation and configuration of the **AWS CLI**, including: `Access Key`, `Secret Key`, and default `Region`.
  * Used AWS CLI to perform basic administrative tasks:
    * Checking account and configuration information.
    * Retrieving the list of Regions.
    * Creating/deleting S3 Buckets, using Amazon SNS.
    * Managing IAM (creating groups, users, adding users).
    * Creating/deleting Access Keys.
    * Basic configuration and lifecycle management of EC2 (running and terminating instances).
* **Cost Management:**
  * Understand how to manage and monitor AWS costs using the Budgets tool.
  * Successfully created and configured various Budget types (`Cost`, `Usage`, `RI`, `Savings Plan`).
  * Understood and performed **resource cleanup procedures** for effective cost control.
* **AWS Support:**
  * Clearly understand the different AWS Support tiers:
    * **Basic:** Free, supports account and billing issues via the support center.
    * **Developer:** $29/month, offers basic architectural guidance and unlimited technical support (created by Root User).
    * **Business:** $100/month, provides use-case guidance, AWS Support API usage, and unlimited support requests from all IAM Users.
    * **Enterprise:** $15,000/month, ensures the most rigorous security and compliance standards, offers comprehensive architectural and cost optimization strategy support, and prioritizes support cases.
  * Proficient with both the AWS Console interface and basic operations via Console and CLI.
